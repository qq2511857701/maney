## 功能一：度数换算

粘贴验光数据，对齐原站选项后调用 API 换算。

### 使用流程

1. **度数换算** Tab → 粘贴文本 →「解析文本」
2. 校对表确认/修改（盒数默认 1）
3. 「换算」→「复制结果」

### 支持的输入格式

**订单行：**
```
-8.25,-1.25,180（1盒）
-3.50，100，179（1盒）
```

**验光单：**
```
R: -10.00 / -2.00 x 175
OD: -5.75 / -1.25 x 10
```

**聊天格式：**
```
右眼(R) 近视525度(-5.25) 散光100度(-1.00) 轴位180
```

## 功能二：批量采购（lensfit 自动加购）

### 使用流程

1. **批量采购** Tab → 粘贴清单 →「解析清单」→ 校对
2. 点击 **复制采购队列**（JSON 写入剪贴板）
3. VPN 开启，浏览器登录 [lensfit.jp](https://www.lensfit.jp/)
4. 安装油猴脚本（见下方）→ 面板「从剪贴板加载」→「开始加购」
5. 完成后自行去购物车结算

### 采购清单格式

每行一条，**产品名与度数用 Tab 或空格分隔**：

```
散光日抛	－3.25,75,180（2盒）
散光日抛	-3.50,225,180（2盒）
舒日散光日抛	－6.50，175，180（3盒）
散光双周	－6.50，175，170（1盒）
```

| 产品别名 | lensfit 商品 |
|----------|-------------|
| 散光日抛 | ワンデー アキュビュー オアシス 乱視用 (JJ1DAOTR) |
| 舒日 / 舒日散光日抛 | ワンデーアキュビューモイスト 乱視用 (JJ1DAMTR) |
| 散光双周 | アキュビューオアシス 乱視用 (JJ2WAOTR) |

度数格式与订单行相同：`球镜,柱镜整数,轴位（N盒）`，柱镜 `75` = `-0.75`，`125` = `-1.25`。

工具会自动对齐各商品在 lensfit 上的合法下拉选项；盒数超过单次上限时会拆成多次加购。

### 安装油猴脚本

1. 浏览器安装 [Tampermonkey](https://www.tampermonkey.net/)
2. 新建脚本，粘贴 [`scripts/lensfit-cart.user.js`](scripts/lensfit-cart.user.js) 全部内容
3. 保存后访问 lensfit.jp，右下角会出现「Lensfit 批量加购」面板

脚本只负责填表并 POST 到购物车，不处理登录和结算。
